# Wind Turbine Performance Simulation Tool

A comprehensive Python-based simulation tool for modeling and analyzing wind turbine performance under various environmental and design conditions.

## Overview

This educational tool simulates wind turbine power generation using fundamental wind energy equations and statistical modeling. It's designed for students, researchers, and professionals in renewable energy to understand how different parameters affect turbine performance.

## Key Features

- **Interactive Parameter Adjustment**: Modify turbine design and environmental parameters in real-time
- **Weibull Wind Distribution**: Generate realistic wind speed data using statistical modeling
- **Power Curve Visualization**: Display the relationship between wind speed and power output
- **Performance Metrics**: Calculate capacity factor, energy production, and efficiency
- **Educational Focus**: Clear explanations of equations and physical principles
